## 耳机电量 v0.2.0

本次版本完成了“长期自用版”收口，主线更稳定，代码结构更清晰，首页保持简洁。

### 主要更新
- 固化正式目标设备主线（41:42:D3:16:6F:68）
- 刷新流程统一为：
  - 连接 -> discoverServices -> 读取 180F/2A19 总电量
  - 启用回包通知
  - 自动发送两条已验证命令
  - 解析 `DD ?? 04 0C XX YY ZZ AA` 分电量帧
- 首页只保留核心信息：
  - 总电量
  - 最近更新时间
  - 当前连接状态
  - 左耳 / 右耳 / 充电仓
  - 刷新电量
  - 导出日志

### 稳定性与维护性
- 清理历史实验残留分支与旧解析器
- 收敛 BLE 会话、目标识别、协议解析、UI 状态模型职责
- 保留必要日志（连接、2A19、写入、回包、解析、导出）

### 兼容与限制
- 当前定位为个人自用单设备主线
- 充电状态语义暂不作为正式 UI 功能
- 不发送未知私有命令，只使用已验证的两条命令
